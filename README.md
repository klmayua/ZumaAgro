# Zuma Agro Allied Integrated Limited Website

This is the official website for Zuma Agro Allied Integrated Limited, a nationwide agro-industrial platform designed to address Nigeria's food security, protein supply, and agro-industrialisation gaps through a structured, secure, and scalable operating model.

## Project Overview

Based on the integrated ranch system partnered with the Nigerian Army and other paramilitary institutions, this website showcases:

- Corporate vision and strategic positioning
- Operations across livestock, grains, poultry, cash crops, and apiculture
- Sustainability and ESG initiatives
- Investor relations and market access
- Technology-enabled agro-industrial development

## Technology Stack

- **Frontend**: Next.js 14 with App Router
- **Styling**: Tailwind CSS with custom design system
- **CMS**: Sanity.io (planned integration)
- **Deployment**: Netlify
- **Animations**: Framer Motion
- **Search**: Algolia (planned integration)

## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/klmayua/ZumaAgro.git
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser

## Deployment

This project is configured for deployment on Netlify. Simply connect your GitHub repository to Netlify for automatic deployments on each push to the main branch.
